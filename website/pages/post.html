Power on self test= Post

Post als erstes die Zentrale Hardware also folgende Komponenten:

  • Cpu
  • Bios-Rom(Bildung einer Prüfsumme)
  • Dma-Kontroller
  • ersten 64k des Rams
  • Interrupt und Cache-Controller

Danach passiert der »Test und Initialisierung von System-Erweiterungen«

  • Ram über 64k
  • Schnittstellen
  • Disketten Festplatten Controler
  • Rom erweiterungen werden gesucht und Initialiesiert zum Beispiel Video-rom

Tritt in dieser Phase ein Fehler auf wird eine Tonfolge (teilweise auch über eine LED mit einer Folge von Kurz an aus und lang an aus) abgespielt. Für bestimmter Fehler sind genaue signale vorgeschrieben:

  • Anhaltend oder Kein ton = Kein Netzteil
  • Einmal Lang = leom DRA;-Refresh
  • Einmal Lang, Einmal Kurz = Motherboard
  • Einmal lang, zweimal Kurz = Video-Controller oder Bildschirmspeicher
  • Dreimal Kurz = RAM
  • einaml kurz laufwerk oder Video-Controller

Bootgeräte

Sind alle Tests vom Post Erfolgreich sucht das Bios nach Bootgeräten für eine gültige Bootsquenz. Bootgeräte können volgende Geräte sein:

  • Diskette
  • Festplatte
  • CD-ROM-Laufwerk
  • Netzwerkkarte

Welche Bootmedien durchsucht werden und die Reihenfolge, in der das BIOS die Geräte durchsucht, kann meistens im CMOS-Setup eingestellt werden. Der erste gefundene Bootkode wird geladen und gestartet.
